This repository is a test for the godot-object-compiler.

To build this repository, the [godot-object-compiler](https://github.com/LucaTuerk/godot-object-compiler) and [godot-object-compiler](https://github.com/godotengine/godot-cpp) repositories must be cloned into the repository root directory. They are not added as submodules as workflows may choose their own versions and branches.

Build using CMake, afterwards the godot project at ./project can be imported into godot (this repository currently tests for Godot 4.5.1).
